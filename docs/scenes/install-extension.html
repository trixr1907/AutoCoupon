1. Erweiterung installieren

Lade die Erweiterung in Chrome oder Firefox und pinne sie an die Symbolleiste. Danach ist das AutoCoupon-Icon immer direkt erreichbar.

Tutorial Schritt 1
https://www.payback.de/coupons
AutoCoupon angepinnt

PAYBACK im Browser

Nach dem Laden der Erweiterung oeffnest du PAYBACK wie gewohnt im Browser und meldest dich normal an.

Coupon-Bereich

Die Erweiterung arbeitet nur lokal auf der sichtbaren Seite.

Browser-Kontext
Kein Extra-Login

Login und Schutzabfragen bleiben komplett in deiner Hand.

Keine Umgehung

Wichtig

AutoCoupon fuegt sich dezent ein. Das Branding bleibt klein, damit die eigentliche Coupon-Aktivierung immer im Vordergrund steht.
