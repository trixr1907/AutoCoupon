🔧 Firefox Installation

Schritt 1: Code kopieren

Klicke auf "Code kopieren" oben oder markiere den kompletten Code und kopiere ihn (Strg+C).

Schritt 2: Lesezeichen erstellen

Firefox:

  • Drücke Strg+Shift+B um die Lesezeichen-Seitenleiste zu öffnen
  • Rechtsklick in die Lesezeichen-Seitenleiste
  • "Neues Lesezeichen..." wählen
  • Name: "Payback Coupon-Klicker"
  • URL: Den kopierten Code hier einfügen
  • Speichern

Schritt 3: Testen

1. Gehe zu payback.de

2. Melde dich an und navigiere zur Coupon-Seite

3. Klicke auf dein Lesezeichen "Payback Coupon-Klicker"

4. Du solltest eine Status-Anzeige oben rechts sehen! 🎯

🔍 Debugging-Hilfen

Falls das Bookmarklet nicht funktioniert:

  1. Firefox Konsole öffnen: F12 → "Konsole"
  2. Bookmarklet ausführen und Fehlermeldungen beachten
  3. Shadow DOM prüfen: Suche in der Konsole nach "Coupon X aktiviert"
  4. Status-Anzeige: Sollte oben rechts erscheinen
